Chrome E2E助手 版本資訊
============================

程式名稱: Chrome E2E助手
版本號: V2.0
發布日期: 2025-07-02
開發者: JOVIAN

版本特性
--------
- 改進的使用者介面，增加明顯的底部按鈕
- 新增字體大小調整功能(A+/A-)，支援從8至24的字體範圍調整
- 優化腳本載入和執行流程，提高穩定性
- 改進錄製和執行過程中的錯誤處理
- 錄製完成後自動驗證腳本功能
- 新增詢問是否立即執行錄製腳本的選項

修復問題
--------
- 修復腳本載入按鈕無法正常工作的問題
- 修復錄製和執行腳本時的錯誤處理
- 解決 AttributeError: 'BrowserAutomation' object has no attribute 'is_paused' 錯誤
- 增強錯誤處理，確保瀏覽器關閉時能優雅退出
- 修復執行腳本按鈕點擊後無響應的問題
- 修復 'str' object has no attribute 'get' 錯誤

系統需求
--------
- Windows 10 或以上版本
- Python 3.8+
- 與系統相容的 Chrome 瀏覽器

安裝依賴
--------
pip install -r requirements.txt
python -m playwright install

使用方法
--------
1. 運行 python main.py 啟動程式
2. 使用「開始錄製」按鈕開始錄製腳本
3. 使用「載入腳本」按鈕載入已有腳本
4. 使用「執行腳本」按鈕執行當前載入的腳本
5. 使用「A+/A-」按鈕調整界面字體大小

常見問題
--------
Q: 為什麼錄製完成後腳本不能執行？
A: 請確保腳本內容有正確載入到編輯區域。若內容為空，請檢查temp_script.txt文件。

Q: 執行時出現 "AttributeError" 錯誤怎麼辦？
A: 此問題已在V2.0版本修復，請確保使用最新版本。

Q: 如何創建自定義測試頁面？
A: 將HTML文件放入web目錄，然後使用「錄製本地頁面」功能選擇該文件。
